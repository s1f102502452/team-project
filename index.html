プログラミングとは？

コンピュータに「こう動いてほしい！」という手順やルールを、
人が読めるかたちの“コード”で書いて伝える作業のこと。

＜プログラミングでできることの代表例＞

  • Webサイト / Webアプリの作成
  • スマホアプリの開発
  • データ分析や機械学習による予測・解析
  • 事務作業の自動化（ファイル整理や定型レポート作成など）
  • 組み込み機器（家電やロボット）の制御
  • ゲームの制作

＜プログラミングってどんな流れで動いているの？＞

  1. プログラミング言語を使って、ソースコードを書く
  2. コンパイラやインタプリタが人間のコードを機械語に翻訳
  3. コンピュータが機械語を実行
  4. 画面に結果が出る

＜おすすめのプログラミング言語＞

８個のパターンに分けて紹介！

１．データ分析・機械学習なら...
python

豊富なライブラリで、すぐにデータ処理を始められる！


２．Webページの動的な動きをつけたいなら...
JavaScript

ブラウザで、その場ですぐ結果を確認できる！


３．大規模業務システムなら...
Java

安定性が高く、企業導入例も多い！


４．Androidアプリ開発なら...
Kotlin

モダン文法で、コードがシンプルに書ける！


５．iOSアプリ開発なら...
Swift

Apple公式言語で、最新機能にも対応しやすい！


６．ゲーム制作なら...
C#（Unity）

無料エンジンUnityで、初心者でも始めやすい！


７．組み込み・ハード制御なら...
C

ハードに近い処理を学べる王道言語！


８．サーバーサイドWebなら...
Ruby（Rails）

文法が分かりやすく、高速にWebアプリを作れる！
