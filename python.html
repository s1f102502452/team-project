Pythonについて

Pythonとは？

Webアプリケーション開発やデータ開発など幅広い分野で汎用の高水準プログラム言語。現在は世界中で最も人気のあるプログラム言語のひとつである。

基本的な関数・文法

  1. 型（type）
    • int型 : 整数
    • float型 : 浮動小数点数（小数）
    • str型 : 文字列
  2. 関数
    • print()

      Pythonに値をコンソール上に表示させることを指示できる

    • input

      ユーザーが文字列を入力することができる

    • int(), float(), str()

      int() : "()"で囲まれたものを整数の型に変換できる
      float() : "()"で囲まれたものを浮動小数点数の型に変換できる
      str() : "()"で囲まれたものを文字列の型に変換できる
