Pythonについて

Pythonとは？

Webアプリケーション開発やデータ開発など幅広い分野で汎用の高水準プログラム言語。
現在は世界中で最も人気のあるプログラム言語のひとつである。

基本的な文法

  1. 変数

    変数とはデータや情報を格納しておく物
    変数は格納されているデータを変更することができる

  2. データ型（type）

    プログラム内で扱うデータの種類や性質を定義するもの
    データ型を理解することで、効率的かつ正確にデータを操作することができる

  3. 型（type）

      例

    • int型 : 整数
    • float型 : 浮動小数点数（小数）
    • str型 : 文字列
  4. 演算子
    • 算術演算子

      値に対して特定の操作を行うための記号やキーワードのことを指す
      計算や比較、論理操作など、さまざまな処理を簡潔に記述するために使用される

      • + : 足し算
      • - : 引き算
      • * : 掛け算
      • / : 割り算
      • // : 割った商の整数値
      • % : 割った時の割り切れなかった余り
      • ** : ある数をｎ回掛けた数（べき乗）
    • 比較演算子
      • == : 2つの値がイコールの時「True」
      • != : 2つの値がイコールでない時「True」
      • > : 左の値が右の値より大きい時「True」
      • < : 左の値が右の値より小さい時「True」
      • >= : 左の値が右の値以上の時「True」
      • <= : 左の値が右の値以下の時「True」
  5. 関数

    関数とは様々な処理が機能の一つとして一つにまとまっているもの
    プログラミングでは、関数を呼び出して様々な機能を使っていく

    • 例
    • print()

      Pythonに値をコンソール上に表示させることを指示できる

    • input()

      ユーザーが文字列を入力することができる

    • int(), float(), str()

      int() : "()"で囲まれたものを整数の型に変換できる
      float() : "()"で囲まれたものを浮動小数点数の型に変換できる
      str() : "()"で囲まれたものを文字列の型に変換できる

  6. 条件文
  7. 構文
    • if 条件A：処理B

      条件Aが成り立つならば処理Bを実行する

  8. 繰り返し
    • for 変数 in オブジェクト:

      オブジェクトにはrange関数やリストなどを使用
      range関数 : rangeで指定された分だけ処理を繰り返す
      リスト : リストの要素を一つずつ取り出して処理を実行

    • while 条件A：処理B

      条件Aが成り立つ間処理Bを繰り返す
