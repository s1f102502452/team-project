Pythonについて

Pythonとは？

Webアプリケーション開発やデータ開発など幅広い分野で汎用の高水準プログラム言語。
現在は世界中で最も人気のあるプログラム言語のひとつである。

基本的な文法

  1. 型（type）
    • int型 : 整数
    • float型 : 浮動小数点数（小数）
    • str型 : 文字列
  2. 演算子
    • 算術演算子
      • + : 足し算
      • - : 引き算
      • * : 掛け算
      • / : 割り算
      • // : 割った商の整数値
      • % : 割った時の割り切れなかった余り
      • ** : ある数をｎ回掛けた数（べき乗）
    • 比較演算子
      • == : 2つの値がイコールの時「True」
      • != : 2つの値がイコールでない時「True」
      • > : 左の値が右の値より大きい時「True」
      • < : 左の値が右の値より小さい時「True」
      • >= : 左の値が右の値以上の時「True」
      • <= : 左の値が右の値以下の時「True」
  3. 関数
    • print()

      Pythonに値をコンソール上に表示させることを指示できる

    • input()

      ユーザーが文字列を入力することができる

    • int(), float(), str()

      int() : "()"で囲まれたものを整数の型に変換できる
      float() : "()"で囲まれたものを浮動小数点数の型に変換できる
      str() : "()"で囲まれたものを文字列の型に変換できる

  4. 構文
    • if 条件A：処理B

      条件Aが成り立つならば処理Bを実行する

    • while 条件A：処理B

      条件Aが成り立つ間処理Bを繰り返す
