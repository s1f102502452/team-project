Java Scriptについて

JavaScriptとは？ - ウェブサイトに「命」を吹き込むプログラミング言語

私たちが日々インターネットで触れるウェブサイトは、単なる情報の羅列ではありません。ボタンをクリックするとメニューが開いたり、画像がスライドしたり、フォームに入力するとリアルタイムでエラーが表示されたり… これらの「動き」や「対話」を実現しているのが、**JavaScript（ジャバスクリプト）**というプログラミング言語です。

基本的な文法

  1. 変数

    変数は、データ（数値、テキストなど）を一時的に保存しておくための「箱」のようなものです。JavaScriptでは、letやconstといったキーワードを使って変数を宣言します。

    • let : 後から値を変更できる変数を宣言する
    • const : 一度値を設定すると、後から変更できない（定数）変数を宣言する
  2. データ型

    JavaScriptでは、扱うデータの種類によって「データ型」があります。主なデータ型には以下のようなものがあります。

    • 文字列(String) : テキストデータ。シングルクォート「'」またはダブルクォート「"」で囲む
    • 数値(Number) : 整数や少数などの数値データ
    • 真偽値(Boolean) : ture(真)またはfalse(偽)の２つの値を持つデータ。条件分岐などで使われる。
    • 配列(Array) : 複数のデータを順序をつけて格納できるリストのようなもの。[] で囲む
    • オブジェクト(Object) : キーと値のペアでデータを格納できる、より複雑なデータ構造。{} で囲む
  3. 演算子
  4. 条件分岐
  5. 繰り返し処理
  6. 関数
